# Cabridge

Cabridge是由MBLC7使用Python开发的一款Minecraft启动器（作者一时兴起写的）

作者也希望能得到很多Star和Fork，觉得不错就支持一下吧！

也可以进一下[Discord](https://discord.gg/aunpuZUFnh)！

## 需不需要下载Python？

**不需要**，因为作者使用Nuitka打包和Microsoft Visual C++编译，实现了打开即用！
